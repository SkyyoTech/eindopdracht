Consistentie en doelgerichtheid

Een website moet consistent blijven en één doel volgen, je kan niet zomaar een link die leidt naar een youtube video die je leuk vind toevoegen.
Als je afbeeldingen toevoegt dan moeten die ook gaan over het doel, je zal ook nooit een mooie tekening van een 3 jaar oud kind op een website voor privévliegtuigen zien.
