Zichtbaarheid en Rangschikking

Je moet niet een achtergrond kiezen wat ervoor zorgt dat je niet meer je tekst gemakkelijk kan lezen, dat zorgt voor heel wat meer rust in de ogen van de websitebezoeker.
Plaatjes zijn beter dan tekst, mensen worden eerder aangetrokken door plaatjes dan een groot blok tekst. Al kan dit niet altijd, is het wel belangrijk om het zo veel mogelijk te doen. Rangschikking.

Sommige informatie is minder belangrijk dan andere, je wilt graag dat de websitebezoeker meteen de belangrijke informatie ziet.
Je kan bijvoorbeeld tekst groter, kleiner of donkerder maken om het aantrekkelijker te maken voor het oog. Je kan ook bij belangrijke afbeeldingen een kader markeren
